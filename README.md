# Finding Aids Rails App
## Overall design
A quick overview of the various "subsystems"
### lib/ead/elements/
This directory contains classes that encapsulate the pertinent EAD elements.
The classes supply class methods that can be used to access pertinent content
(such as attributes or enclosed child elements) using
Nokogiri::XML::Searchable.xpath. Please note that just a subset of the elements
specified by EAD are implemented; furthermore, for the imeplemented elements,
only a subset of the possible content (including children elements and
attributes) are processed -- thus the use of "pertinent" in the previous
sentences.
### lib/archive_space/parsers/
This directory contains classes that use the EAD-related classes described
above to parse out content from the EAD XML generated by ArchiveSpace. Each
class provides accessor attributes (may be changed to read-only attributes_
to access the parsed out information.
